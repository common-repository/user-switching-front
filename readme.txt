== Description ==

WordPress plugin that allows to use the wonderful plugin User Switching directly from front-end by using admin bar.

== Requirements ==

* PHP 5.4
* <a title="Download plugin User Switching" href="https://fr.wordpress.org/plugins/user-switching/">User Switching</a>

*  Latest stable version : https://github.com/TweetPressFr/user-switching-front

== Installation ==

1. Upload plugin files to the /wp-content/plugins/ directory
2. Activate the plugin through the Plugins menu in WordPress
2. Be sure to have the plugin User Switching activated otherwise, indeed, it won't do anything
